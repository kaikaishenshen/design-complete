# Distill 流程

剥离到核心本质，移除复杂性。目标是减少至*最低可行优雅*。

## 何时使用

用户在说："这太复杂了。简化它。"

## 怎么做

1. **识别每个元素的目的。** 如果它不服务于用户目标，移除它。
2. **应用精简操作：**
   - 将 3 步流程合并为 2 步
   - 从卡片中移除装饰性图标
   - 将两个部分合并为一个
   - 移除眼眉标和编号标记
   - 将多列改为单列布局
   - 将复杂的 CTA（主要、次要、三级）减少为单一操作
3. **检查是否失去本质。** 当你移除东西时，核心信息是否仍然清晰地传达？

## 原则

- 精简是关于去除*干扰*，而不是去除*意义*。
- 问：这个元素*必须*存在吗？如果答案是不是，移除它。
- 限制行动号召。一次只要求用户做一件事。
