# Bolder 流程

安全或平淡的设计需要更多胆量。目标是**推过舒适区**——不破坏它，而是推它。

## 何时使用

用户在说："这很安全。让它更大胆。"

## 怎么做

1. **分析现有内容。** 找出安全的地方在哪里。
2. **应用大胆操作：**
   - **排版**：将 hero 标题增大一档（如果 < 4rem，尝试 5-6rem）。为展示文本添加黑体或粗体字重。
   - **色彩**：将主题色饱和度提高。将强调色使用增加 50%。
   - **布局**：重叠一个元素。使用跨越 2 列不对称功能。添加全出血区域。
   - **间距**：在某些区域加倍，在另一些区域减半以创造节奏。
   - **动效**：如果缺少动效，添加滚动触发的揭示。加强交错。
3. **检查是否会过度，则回退。** 如果您应用的更改感觉像是现在需要`/impeccable quieter`，那就去掉一点。

## 原则

- 更大胆 ≠ 更嘈杂。更大胆意味着*更自信*。
- 一个大胆的举动胜过三个小调整。
- 不要一次把每个旋钮都调大。选择一两个杠杆并真正推动它们。
