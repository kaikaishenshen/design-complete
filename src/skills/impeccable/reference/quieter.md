# Quieter 流程

过于激进或过度刺激的设计需要降低音量。目标是**在不过度稀释的情况下抚平边缘**。

## 何时使用

用户在说："这太花哨了。让它安静点。"

## 怎么做

1. **分析过度刺激点。** 哪里感觉过于大声？
2. **应用安静操作：**
   - **排版**：降低最大标题字号。减少字重对比度。
   - **色彩**：降低主题色饱和度。减少强调色表面覆盖。
   - **布局**：增加元素之间的间距。简化网格结构。
   - **动效**：减少动画持续时间。用交叉淡入淡出替换弹跳效果。
   - **间距**：在部分之间增加空白。为内容提供呼吸空间。
3. **检查是否变得平淡。** 如果在降低音量后感觉无聊，保留一个单一的大胆元素作为锚点。

## 原则

- 安静 ≠ 无聊。安静意味着*沉稳*。
- 负空间是朋友。更多的空白创造了宁静。
- 当有疑问时，移除一个元素。少即是多。
