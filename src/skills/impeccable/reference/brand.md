# 品牌注册参考

当项目是**品牌注册**时加载：设计是产品本身（不是服务于产品的 UI）。这适用于落地页、活动页、品牌站、作品集、长文内容。这里的规则补充了主 SKILL.md 中的通用规则。

## 品牌工作流的不同之处

### 排版

- 品牌站点可以使用**一个展示字体**，对于内容站点使用**一个正文字体**。在品牌注册中，一个字体家族带多种字重（包括 display weight 如 Extrabold、Black）是强有力的工具。
- `clamp()` 最大值上限：6rem。hero 标题可以接近上限；品牌注册在视觉上有更大的发挥空间。
- 按比例尺寸缩放：`h1: 3.5rem, h2: 2.5rem, h3: 1.75rem`。视觉层级创建叙事流动性。

### 色彩策略

品牌注册可以采用四种色彩策略之一：

- **克制**：有色中性色 + 一个 ≤ 10% 的强调色。简洁、知性。
- **投入**：一个饱和色覆盖 30-60% 的表面。大胆自信。
- **全调色板**：3-4 个命名角色，每个被刻意使用。丰富。
- **浸染**：表面本身就是颜色。沉浸式的品牌声明。

### 空间设计

- 品牌站点有更大的空间许可。在部分之间使用宏空白（`padding: clamp(4rem, 10vh, 8rem)`）。
- 不对称布局：网格中跨越 2 列的元素。视觉兴趣点。

### 品牌特定反模式

除了通用绝对禁令之外，品牌注册还要避免：

- **奶油色 SaaS 风格**。奶油/浅色沙色/淡黄褐色正文背景是每个 AI 工具的默认，即使在工作流名称中重命名也仍然是奶油色。
- **通用创业素材**。在 hero 背景中的模糊办公桌和咖啡，人使用笔记本电脑的库存照片。
- **英雄-指标模板**。大数字上方有简短标签和紫色强调色。SaaS 陈词滥调。
- **不承诺的色彩**。带有浅色变体按钮的小心调色板。为你的品牌挺身而出。

### 类似注册的参考

- 品牌注册共享品牌行为，因此 `reference/product.md` 规则不适用。
- 但是，品牌站点的内部页面（定价、常见问题、公司）可以从产品化的敏感性中受益。如果内部部分的 UX 看起来像页面其余部分一样有广告感，它就会感到不一致。在这些区域应用间距和清晰度的产品注册规则，但保持品牌视觉处理。

### 区域类型

**落地页**（主页、活动页、产品发布）：
- 通过单个滚动叙事引导用户
- 每个部分提供足够的视觉对比以保持兴趣
- 最后的 CTA 是强制性

**内容页面**（关于、博客、资源）：
- 可读性优先于独创性
- 宏空白；阅读中的间距是慷慨的
- 印刷式层次创建扫读路径

**作品集**（工作室/代理/创意站点）：
- 作品是主角——让界面退后
- 全出血布局，大量负空间
