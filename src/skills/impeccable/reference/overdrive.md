# Overdrive 流程

突破常规技术极限。适合需要展示技术实力的项目——既高品质又技术惊艳。

## 何时使用

用户在说：「让它惊艳」或「展示你们能做什么」。

## 怎么做

- **高级动画**：Three.js 集成、视差滚动、WebGL
- **页面过渡**：在页面之间使用 morphing 动画
- **鼠标追踪**：光标微妙的视差、跟随 CTA 的粒子
- **滚动叙事**：由滚动驱动的全页故事——同屏滚动、视差章节
- **音频/视觉**：微妙的背景环境音、数据驱动的动画画布

## 原则

- 高级效果必须加载快，不能影响核心 UX
- 为不支持高级功能的浏览器提供优雅降级
- 仅在品牌注册中使用（产品注册以效率和清晰度优先）
- 性能仍然重要——测试在慢速设备上是否流畅
